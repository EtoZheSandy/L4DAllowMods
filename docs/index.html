Workshop utility for Left 4 Dead 2

Enable selected addons with a clean desktop workflow.

L4D Allow Mods prepares Workshop .vpk files, updates the game's gameinfo.txt, and gives you quick controls for enabling, disabling, searching, and restoring your addon setup.

Built for quick mod sessions

Everything important stays one click away.

Addon control

Select the Workshop addons you want, enable them before launch, and disable them when you are done.

Workshop scanning

The app reads your addons/workshop folder and shows addon titles, previews, descriptions, and links.

Gameinfo restore

Refresh cached gameinfo.txt data, delete addon cache, and restore saved settings from the Settings screen.

No runtime hacks

The workflow is file-based. There is no DLL injection, no memory patching, and no separate launcher.

Application screens

A dark, focused UI for managing L4D2 addons.

Get the app

Download the latest Windows build from GitHub Releases.

Install the MSI or EXE build, choose your Left 4 Dead 2 folder, select the addons you want, and enable the prepared setup before launching the game.
